# coolWeather
借鉴郭霖大神的第一行代码来做的，添加了一些新东西，可以给一些想做天气软件的新手一些思路。

## 用到的技术及框架 ##
- sqlite数据库操作
- 安卓网路异步
- json数据解析
- fragment的运用
- android-univeral-imagload框架
- 数据存储
- 下拉刷新
- 侧滑菜单
- 状态栏一体化
